Event Time Calculator

Privacy Policy

Last updated: 2025-12-13

Event Time Calculator is a Chrome extension that calculates total time spent on events in Google Calendar. This extension is read-only and does not modify calendar data.

Data We Access

With your permission, the extension accesses Google Calendar event data needed to perform calculations:

  • Event titles
  • Event start and end times (including all-day events)

How We Use Data

Data is used only to calculate durations and totals that you explicitly request.

  • Compute event duration (start â†’ end)
  • Compute total time across matching events for the selected time range

OAuth Scope

The extension requests the following Google API scope: https://www.googleapis.com/auth/calendar.readonly

Processing, Storage, and Sharing

  • Local processing: calculations run locally in your browser.
  • No server storage: we do not store your calendar data on external servers.
  • No selling/sharing: data is not sold or shared with third parties.
  • No analytics/tracking: the extension does not use analytics, tracking pixels, or ad networks.

Data Retention

The extension does not retain your Google Calendar event data outside of runtime memory needed to complete a calculation.

Security

We follow least-privilege access (read-only scope) and do not transmit calendar data to any external backend. You can revoke access at any time from your Google Account permissions.

Your Choices

You can uninstall the extension at any time or revoke access via your Google Account settings.
