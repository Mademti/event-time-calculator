Privacy Policy

Last updated: 2025-12-16

Event Time Calculator is a Chrome extension that helps you calculate total time spent on events in Google Calendar. The extension is designed to be privacy-first and read-only.

1. Data We Access

When you choose to use the calculator, the extension may access Google Calendar data required for the calculation:

  • Event titles
  • Event start and end times (including all-day events)

2. How We Use Data

Data is used only to compute totals that you explicitly request:

  • Compute duration for each matching event
  • Sum durations across the selected time range

3. OAuth Scope

The extension requests Google Calendar read-only access:
https://www.googleapis.com/auth/calendar.readonly

4. Processing, Storage, and Sharing

  • Local processing: calculations run in your browser.
  • No server storage: we do not store your calendar data on any server.
  • No selling or sharing: we do not sell or share your data with third parties.

5. Data Retention

The extension does not retain your Google Calendar data beyond the temporary runtime memory needed to complete the calculation.

6. Security

We follow least-privilege access (read-only scope). You can revoke access anytime from your Google Account permissions.
