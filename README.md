# Event Time Calculator

Event Time Calculator is a Chrome extension that helps you calculate event durations and total hours directly inside Google Calendar.

## Overview

This extension adds a lightweight panel to Google Calendar that allows you to:
- Calculate the duration of individual events
- Sum total hours across multiple events or a selected date range
- Quickly understand how your time is allocated

Designed to be simple, fast, and non-intrusive.

Event Time Calculator accesses Google Calendar data only to perform time calculations requested by the user.

- No data is sold
- No data is shared with third parties
- No server-side storage of calendar data

All processing happens locally in the browser.
