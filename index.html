Chrome Extension Google Calendar Read-only

Track time spent on Google Calendar events

Event Time Calculator is a Chrome extension that helps users calculate the total time spent on Google Calendar events, including recurring events, over a selected date range.

How it works

The extension connects to the userâ€™s Google Calendar using read-only access (calendar.readonly) to retrieve event information selected by the user and compute total durations (for example: total hours across recurring meetings).

What the extension does

  • Reads calendar events selected by the user
  • Expands recurring events to occurrences
  • Calculates total duration for a chosen period
  • Displays aggregated results in the extension UI

What the extension does not do

  • Does not create, edit, or delete any events
  • Does not store Google Calendar data
  • Does not send data to external servers

Data processing & permissions

All calculations are performed locally in the browser. The only Google scope requested is calendar.readonly, which is required to read event details for time calculations.

Read-only

No event is modified or deleted.

Local processing

No data is sent to any server.

Minimal permissions

Uses calendar.readonly only.
