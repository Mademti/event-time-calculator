Event Time Calculator

Calculates the total time spent on a specific event name in Google Calendar across a selected time range (today, next 7 days, this month, this year).

Read-only
Does not create, edit, or delete events.
Local processing
All calculations run in your browser.
No tracking
No analytics, no data selling.

Permissions

Uses Google Calendar read-only access (calendar.readonly) to read event titles and start/end times only when you request a calculation.
